# Основная информация о проекте

Это django приложение, которое будет состоять из функционала для управления ассортиментом магазина(товарами для пикника) и страницами, содержаими данный ассортимент.

# Основной функционал проекта
1. Удобное управление каталогами товаров сайта
2. Понятная сортировка товаров по подходящим категориям
3. Страницы с информацией об отдельно выбранных товарах
4. Четкая навигация по страницам сайта

# Список актуальных страниц магазина

1. Главная страница
- содержит ссылку на станицу с информацией о магазине
- содержит ссылку на станицу с информацией об авторе

2. Страница об авторе
- содержит основную информацию об авторе проекта

3. Страница о магазине
- содержит информацию о теме магазина
